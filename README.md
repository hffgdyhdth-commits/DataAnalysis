# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Metacritic Games Dataset " в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Данные содержат информацию о видеоиграх с сайта Metacritic, включая рейтинги критиков и пользователей, жанры, платформы, разработчиков и даты выпуска. На их основе можно решать задачи анализа факторов успеха игр, сегментации рынка и оценки репутации студий.  
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
